# Tournament Bracket (`tourney.html`)

Double-elimination bracket manager with SVG bracket rendering and score reporting modal.

## Setup

- Enter `4` to `64` names (one per line).
- Optional random seeding.
- Generate winners/losers brackets with finals pathing.

## Match Flow

- Click a match to report score.
- Confirm winner progression and loser drop.
- Bracket state persists in local storage.

## Controls

- `New Bracket`
- `Auto-Fit`
- Portal link + accent/background controls

## Tips

- Use consistent team naming before generation to reduce mid-bracket edits.
- Keep this page as a dedicated display tab when running tournament sessions.
